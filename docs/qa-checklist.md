# Reflix QA 체크리스트 (일반 사용자 관점)

> 일반 사용자가 사이트를 처음 방문해서 모든 기능을 사용하는 시나리오 기준

---

## 1. 랜딩 페이지 (`/ko`)

### 네비게이션
- [ ] 로고 클릭 → 홈으로 이동
- [ ] "탐색하기" 버튼 → `/browse`로 이동
- [ ] "요금제" 링크 → 페이지 내 요금제 섹션으로 부드러운 스크롤
- [ ] 로그인 버튼 → `/login`으로 이동 (비로그인 상태)
- [ ] 계정 버튼 → `/account`로 이동 (로그인 상태)

### 히어로 섹션
- [ ] 배경 클립 MP4 그리드 자동재생 (소리 없이)
- [ ] "지금 탐색하기" CTA → `/browse`로 이동
- [ ] 카테고리 필 태그 표시 확인 (전투, 감정, 연출 등)

### 기능 소개
- [ ] 3개 기능 행의 비디오 자동재생
- [ ] 프레임 재생 섹션의 VideoPlayer 실제 조작 가능 (속도 조절, 재생/정지)
- [ ] 스크롤 진입 시 fade-in 애니메이션 동작

### 통계
- [ ] 클립 수/태그 수 카운트업 애니메이션 (화면 진입 시 0→목표값)

### 요금제 프리뷰
- [ ] 월간/연간 토글 전환 동작
- [ ] 연간 선택 시 -17% 할인 표시
- [ ] Free "무료로 시작" → `/browse`
- [ ] Pro "Pro 시작하기" → 요금제 모달 열림

## 2. 로그인 (`/ko/login`)

### 매직 링크
- [ ] 이메일 입력 + "매직 링크 보내기" 클릭
- [ ] 전송 중: 버튼 비활성화 + 스피너
- [ ] 전송 성공: "이메일을 확인하세요" 화면 전환
- [ ] 다른 탭에서 매직 링크 클릭 → 원래 탭 자동 리다이렉트
- [ ] 잘못된 이메일 형식 입력 시 에러 처리

### Google OAuth
- [ ] "Google로 계속하기" 클릭 → Google 인증 플로우
- [ ] 인증 완료 후 콜백 → `/browse`로 리다이렉트

### 에러 처리
- [ ] `?error=auth` 파라미터 시 실패 메시지 표시
- [ ] `?error=replaced` 시 "다른 탭에서 재로그인" 메시지

## 3. 탐색 (`/ko/browse`)

### 초기 진입
- [ ] 첫 방문 시 브랜드 스플래시 표시 후 자동 전환
- [ ] 피드 뷰가 기본 상태로 표시

### 네비바
- [ ] 검색바에 텍스트 입력 → 실시간 필터링 (한국어 초성 검색 지원)
- [ ] 영타→한글 자동 변환 검색 (예: `wjsfl` → `전투`)
- [ ] 검색어 입력 후 URL에 반영 확인
- [ ] 언어 전환 (KO↔EN) 시 필터 상태 유지
- [ ] 다크/라이트 테마 토글 동작
- [ ] 좌우 패널 토글 버튼 동작

### 좌측 패널
- [ ] 콘텐츠 모드 토글: 전체 / 연출 / 게임
- [ ] "둘러보기" 클릭 → 모든 필터 초기화 + 피드 뷰 복원
- [ ] "모든 태그" 클릭 → 태그 필터 패널 열기
- [ ] 폴더 클릭 → 해당 폴더 필터 적용 (마소니 뷰 전환)
- [ ] Cmd+클릭으로 여러 폴더 다중 선택
- [ ] 폴더 트리 펼치기/접기
- [ ] 폴더 옆 클립 수 표시 확인

### 피드 뷰
- [ ] 카테고리별 섹션 헤더 + 클립 수 표시
- [ ] "전체 보기 →" 클릭 → 해당 폴더 필터 + 마소니 뷰 전환
- [ ] 히어로 클립 viewport 진입 시 자동재생

### 마소니 그리드
- [ ] 클립 카드 단계별 로딩: 블러(LQIP) → 썸네일(WebP) → 미리보기(MP4)
- [ ] 호버 시 태그/재생시간 정보 표시
- [ ] 싱글 클릭 → 클립 선택 (파란 테두리 + 우측 패널 표시)
- [ ] 더블 클릭 → QuickView 모달 열기
- [ ] 대량 클립 스크롤 시 버벅임 없음 (가상화 확인)
- [ ] 검색/필터 결과 수 상단 표시

### 썸네일 크기 조절
- [ ] 슬라이더로 그리드 컬럼 수 조정
- [ ] `-` / `+` 키보드 단축키 동작

### 태그 필터 패널
- [ ] 태그 검색 입력 → 실시간 필터링
- [ ] 좌측 그룹 클릭 → 우측에 해당 그룹 태그만 표시
- [ ] 좌클릭: 태그 포함 필터 (unselected → included → unselected)
- [ ] 우클릭: 태그 제외 필터 (unselected → excluded → unselected)
- [ ] 제외 태그 빨간색 취소선 표시
- [ ] 태그별 클립 수 표시
- [ ] ESC 또는 바깥 클릭으로 닫힘

### 서브 툴바 필터 뱃지
- [ ] 적용 중인 필터 뱃지 표시 (폴더/태그)
- [ ] 뱃지 클릭 → 해당 필터 즉시 제거
- [ ] 넘치는 뱃지 "+N개" 오버플로우 표시

## 8. 요금제 모달 (PricingModal)

> `/pricing` 독립 페이지 삭제됨 → 전역 모달로 전환

### 진입점 (4곳에서 모달 열림)
- [ ] 랜딩 요금제 섹션: Pro "Pro 시작하기" 클릭 → 모달 열림
- [ ] 계정 페이지: Free 유저 "Pro로 업그레이드" 클릭 → 모달 열림
- [ ] Browse 배너: "Pro로 잠금 해제" 클릭 → 모달 열림
- [ ] 잠긴 ClipCard: 싱글 클릭 / 더블 클릭 / Enter·Space 키 → 모달 열림

### 모달 동작
- [ ] ESC 키로 닫힘
- [ ] 배경(backdrop) 클릭으로 닫힘
- [ ] 열릴 때 body 스크롤 잠금, 닫히면 복원
- [ ] 포커스 트랩: 열릴 때 포커스 저장, 닫히면 이전 요소로 복원
- [ ] `role="dialog"` + `aria-modal="true"` 접근성

### 요금제 카드
- [ ] 월간/연간 토글 전환 (기본값: 월간)
- [ ] 월간: Pro ₩9,900 / 연간: Pro ₩99,000 (월 ₩8,250 환산)
- [ ] 연간 선택 시 -17% 할인 뱃지 표시
- [ ] Free "무료로 계속" → 모달 닫힘 (현재 화면 유지)
- [ ] Pro "구독 시작" (비로그인) → 모달 닫힘 + `/login`으로 이동
- [ ] Pro "구독 시작" (Free 로그인) → Stripe Checkout 페이지 이동
- [ ] 결제 준비 중: "결제 준비 중..." 로딩 상태
- [ ] 이미 Pro인 경우 → "구독 중" 버튼 비활성화

### 레거시 URL 리다이렉트
- [ ] `/ko/pricing` 접근 → 301 리다이렉트 → `/ko/browse`
- [ ] `/en/pricing` 접근 → 301 리다이렉트 → `/en/browse`

### 반응형
- [ ] 모바일: 세로 스크롤 가능 (`max-h-[calc(100vh-2rem)]`)
- [ ] 데스크탑(md+): 2열 가로 레이아웃

## 9. 무료 vs Pro 제한 (접근 제어)

- [ ] Free: 5개 클립만 접근 가능, 나머지 자물쇠+흐림 처리
- [ ] Free: 잠긴 클립 클릭 → 요금제 모달 열림 (클립 선택 안 됨)
- [ ] Free: 검색/필터 결과도 5개 제한
- [ ] Free: 셔플 버튼 숨김
- [ ] Free: 다중 필터 축 조합 1개만
- [ ] Pro: 전체 클립 접근, 셔플 표시, 필터 무제한
