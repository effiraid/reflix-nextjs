# Reflix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a game animation reference platform (Reflix) with Eagle-style 3-panel browsing, Pinterest masonry grid, animated WebP thumbnails, and an Eagle-to-R2 export pipeline.

**Architecture:** Next.js 16 SSG/ISR hybrid with static JSON data, Zustand state management, and Cloudflare R2 for media hosting. The app uses a `[lang]` dynamic segment for i18n (ko/en), `proxy.ts` for locale redirect, and `next-themes` for light/dark mode. All components live under `src/` with `@/*` path alias.

**Tech Stack:** Next.js 16, React 19, TypeScript 5, Tailwind CSS 4, Zustand, @tanstack/react-virtual, next-themes, Pagefind, @aws-sdk/client-s3

**Spec:** `docs/superpowers/specs/2026-03-22-reflix-design.md`

**CRITICAL — Next.js 16 Breaking Changes:**
- `params` and `searchParams` are **Promises** — always `await` them
- Middleware is now `proxy.ts` (not `middleware.ts`)
- ISR uses `'use cache'` + `cacheLife()` — NOT the old `revalidate` export
- Route type helpers `PageProps<'/[lang]'>` are auto-generated globally
- Read `node_modules/next/dist/docs/` before writing any route file

```typescript
import { NextRequest, NextResponse } from "next/server";
import { LOCALES, DEFAULT_LOCALE } from "@/lib/constants";

export function proxy(request: NextRequest) {
  const { pathname } = request.nextUrl;

  // Skip internal paths
  if (pathname.startsWith("/_next") || pathname.startsWith("/api")) {
    return NextResponse.next();
  }

  // Check if locale is already in path
  const pathnameHasLocale = LOCALES.some(
    (locale) => pathname.startsWith(`/${locale}/`) || pathname === `/${locale}`
  );

  if (pathnameHasLocale) return NextResponse.next();

  // Detect preferred locale from Accept-Language header
  const acceptLanguage = request.headers.get("accept-language") ?? "";
  const preferred = acceptLanguage.includes("ko") ? "ko" : DEFAULT_LOCALE;

  // Redirect to locale-prefixed path
  const locale = LOCALES.includes(preferred as any) ? preferred : DEFAULT_LOCALE;
  return NextResponse.redirect(
    new URL(`/${locale}${pathname === "/" ? "" : pathname}`, request.url)
  );
}

export const config = {
  matcher: ["/((?!_next|api|favicon.ico|.*\\..*).*)"],
};
```

- [ ] **Step 6: Create filter logic `src/lib/filter.ts`**

```typescript
import type { ClipIndex, SortBy } from "./types";

export interface FilterState {
  selectedFolders: string[];
  selectedTags: string[];
  starFilter: number | null;
  searchQuery: string;
  sortBy: SortBy;
  category: string | null;
}

export function filterClips(
  clips: ClipIndex[],
  filters: FilterState
): ClipIndex[] {
  let result = clips;

  // Category filter
  if (filters.category) {
    result = result.filter((c) => c.category === filters.category);
  }

  // Folder filter
  if (filters.selectedFolders.length > 0) {
    result = result.filter((c) =>
      c.folders.some((f) => filters.selectedFolders.includes(f))
    );
  }

  // Tag filter (AND logic — clip must have ALL selected tags)
  if (filters.selectedTags.length > 0) {
    result = result.filter((c) =>
      filters.selectedTags.every((t) => c.tags.includes(t))
    );
  }

  // Star filter
  if (filters.starFilter !== null) {
    result = result.filter((c) => c.star >= filters.starFilter!);
  }

  // Text search (simple substring match on name + tags)
  if (filters.searchQuery) {
    const q = filters.searchQuery.toLowerCase();
    result = result.filter(
      (c) =>
        c.name.toLowerCase().includes(q) ||
        c.tags.some((t) => t.toLowerCase().includes(q))
    );
  }

  // Sort
  return sortClips(result, filters.sortBy);
}

function sortClips(clips: ClipIndex[], sortBy: SortBy): ClipIndex[] {
  const sorted = [...clips];
  switch (sortBy) {
    case "newest":
      return sorted; // index.json is pre-sorted by newest
    case "rating":
      return sorted.sort((a, b) => b.star - a.star);
    case "name":
      return sorted.sort((a, b) => a.name.localeCompare(b.name, "ko"));
  }
}
```

```typescript
"use client";

import { useEffect, useRef } from "react";
import { useSearchParams, useRouter, usePathname } from "next/navigation";
import { useFilterStore } from "@/stores/filterStore";
import type { SortBy } from "@/lib/types";

export function useFilterSync() {
  const searchParams = useSearchParams();
  const router = useRouter();
  const pathname = usePathname();
  const store = useFilterStore();
  // Guard against infinite URL↔Store sync loop
  const isUrlUpdate = useRef(false);
  const isStoreUpdate = useRef(false);

  // URL → Zustand (on mount and browser back/forward)
  useEffect(() => {
    if (isStoreUpdate.current) {
      isStoreUpdate.current = false;
      return; // Skip — this URL change came from the store
    }

    isUrlUpdate.current = true;
    const category = searchParams.get("category");
    const tags = searchParams.getAll("tag");
    const star = searchParams.get("star");
    const sort = searchParams.get("sort") as SortBy | null;
    const folders = searchParams.getAll("folder");

    useFilterStore.setState({
      category: category || null,
      selectedTags: tags,
      starFilter: star ? Number(star) : null,
      sortBy: sort || "newest",
      selectedFolders: folders,
    });
    // Reset after microtask to allow the store effect to see the flag
    queueMicrotask(() => { isUrlUpdate.current = false; });
  }, [searchParams]);

  // Zustand → URL (on filter change from user interaction)
  useEffect(() => {
    if (isUrlUpdate.current) return; // Skip — this store change came from URL

    isStoreUpdate.current = true;
    const params = new URLSearchParams();
    if (store.category) params.set("category", store.category);
    store.selectedTags.forEach((t) => params.append("tag", t));
    store.selectedFolders.forEach((f) => params.append("folder", f));
    if (store.starFilter !== null)
      params.set("star", String(store.starFilter));
    if (store.sortBy !== "newest") params.set("sort", store.sortBy);

    const qs = params.toString();
    const newPath = qs ? `${pathname}?${qs}` : pathname;
    router.replace(newPath, { scroll: false });
  }, [
    store.category,
    store.selectedTags,
    store.selectedFolders,
    store.starFilter,
    store.sortBy,
    pathname,
    router,
  ]);
}
```

```typescript
"use client";

import { useEffect, useRef, useState } from "react";

type LoadStage = "lqip" | "thumbnail" | "webp";

export function useIntersectionLoader() {
  const ref = useRef<HTMLDivElement>(null);
  const [stage, setStage] = useState<LoadStage>("lqip");
  const [isInView, setIsInView] = useState(false);
  const enterTime = useRef<number>(0);

  useEffect(() => {
    const el = ref.current;
    if (!el) return;

    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          setIsInView(true);
          enterTime.current = Date.now();
          // Load thumbnail immediately
          setStage((prev) => (prev === "lqip" ? "thumbnail" : prev));
          // After 300ms in viewport, load webp
          const timer = setTimeout(() => {
            setStage("webp");
          }, 300);
          return () => clearTimeout(timer);
        } else {
          setIsInView(false);
          // Unload webp when out of viewport
          setStage("thumbnail");
        }
      },
      { rootMargin: "200px" }
    );

    observer.observe(el);
    return () => observer.disconnect();
  }, []);

  return { ref, stage, isInView };
}
```

Implements column-based masonry with `@tanstack/react-virtual` for virtualization. Items are distributed across columns by shortest-column-first algorithm, then each column is virtualized independently.

```typescript
"use client";

import { useMemo, useRef, useCallback } from "react";
import { useVirtualizer } from "@tanstack/react-virtual";
import { ClipCard } from "./ClipCard";
import { useClipStore } from "@/stores/clipStore";
import { BREAKPOINTS, GRID_COLUMNS } from "@/lib/constants";
import type { ClipIndex } from "@/lib/types";

interface MasonryGridProps {
  clips: ClipIndex[];
}

function useColumnCount() {
  // This will be refined with a resize observer.
  // For now, default to desktop.
  if (typeof window === "undefined") return GRID_COLUMNS.desktop;
  const width = window.innerWidth;
  if (width < BREAKPOINTS.mobile) return GRID_COLUMNS.mobile;
  if (width < BREAKPOINTS.tablet) return GRID_COLUMNS.tablet;
  return GRID_COLUMNS.desktop;
}

export function MasonryGrid({ clips }: MasonryGridProps) {
  const parentRef = useRef<HTMLDivElement>(null);
  const columnCount = useColumnCount();

  // Distribute clips into columns by shortest-column-first
  const columns = useMemo(() => {
    const cols: ClipIndex[][] = Array.from({ length: columnCount }, () => []);
    const heights = new Array(columnCount).fill(0);

    for (const clip of clips) {
      const shortestCol = heights.indexOf(Math.min(...heights));
      cols[shortestCol].push(clip);
      // Estimate height based on aspect ratio (assume 1 unit width per column)
      heights[shortestCol] += clip.height / clip.width;
    }
    return cols;
  }, [clips, columnCount]);

  return (
    <div ref={parentRef} className="flex gap-3 p-3">
      {columns.map((colClips, colIndex) => (
        <MasonryColumn key={colIndex} clips={colClips} />
      ))}
    </div>
  );
}

function MasonryColumn({ clips }: { clips: ClipIndex[] }) {
  const parentRef = useRef<HTMLDivElement>(null);

  const virtualizer = useVirtualizer({
    count: clips.length,
    getScrollElement: () =>
      parentRef.current?.closest("[data-masonry-scroll]") as HTMLElement | null,
    estimateSize: (index) => {
      const clip = clips[index];
      // Estimate pixel height: column width ~250px × aspect ratio + gap
      return Math.round(250 * (clip.height / clip.width)) + 12;
    },
    overscan: 5,
  });

  return (
    <div ref={parentRef} className="flex-1 min-w-0">
      <div
        style={{ height: virtualizer.getTotalSize(), position: "relative" }}
      >
        {virtualizer.getVirtualItems().map((virtualItem) => {
          const clip = clips[virtualItem.index];
          return (
            <div
              key={clip.id}
              style={{
                position: "absolute",
                top: 0,
                left: 0,
                width: "100%",
                transform: `translateY(${virtualItem.start}px)`,
              }}
              className="pb-3"
            >
              <ClipCard clip={clip} />
            </div>
          );
        })}
      </div>
    </div>
  );
}
```

## Task 12: VideoPlayer + Download Prevention

**Files:**
- Create: `src/components/clip/VideoPlayer.tsx`

- [ ] **Step 1: Create `src/components/clip/VideoPlayer.tsx`**

Custom video player with download prevention built in:

```typescript
"use client";

import { useRef, useState, useCallback } from "react";
import { MEDIA_BASE_URL } from "@/lib/constants";

interface VideoPlayerProps {
  videoUrl: string;
  thumbnailUrl: string;
  duration: number;
  compact?: boolean; // For right panel mini player
}

export function VideoPlayer({
  videoUrl,
  thumbnailUrl,
  duration,
  compact = false,
}: VideoPlayerProps) {
  const videoRef = useRef<HTMLVideoElement>(null);
  const [isPlaying, setIsPlaying] = useState(false);
  const [currentTime, setCurrentTime] = useState(0);
  const [playbackRate, setPlaybackRate] = useState(1);

  const fullUrl = `${MEDIA_BASE_URL}${videoUrl}`;

  const togglePlay = useCallback(() => {
    const video = videoRef.current;
    if (!video) return;
    if (video.paused) {
      video.play();
      setIsPlaying(true);
    } else {
      video.pause();
      setIsPlaying(false);
    }
  }, []);

  const handleTimeUpdate = useCallback(() => {
    if (videoRef.current) {
      setCurrentTime(videoRef.current.currentTime);
    }
  }, []);

  const handleSeek = useCallback(
    (e: React.MouseEvent<HTMLDivElement>) => {
      const video = videoRef.current;
      if (!video) return;
      const rect = e.currentTarget.getBoundingClientRect();
      const ratio = (e.clientX - rect.left) / rect.width;
      video.currentTime = ratio * duration;
    },
    [duration]
  );

  const cycleSpeed = useCallback(() => {
    const speeds = [0.25, 0.5, 1, 1.5, 2];
    const next = speeds[(speeds.indexOf(playbackRate) + 1) % speeds.length];
    setPlaybackRate(next);
    if (videoRef.current) videoRef.current.playbackRate = next;
  }, [playbackRate]);

  const formatTime = (t: number) => {
    const s = Math.floor(t % 60);
    const m = Math.floor(t / 60);
    return `${m}:${s.toString().padStart(2, "0")}`;
  };

  return (
    <div className="relative group">
      {/* Transparent overlay to block direct video interaction */}
      <div
        className="absolute inset-0 z-10"
        onClick={togglePlay}
        onContextMenu={(e) => e.preventDefault()}
      />

      <video
        ref={videoRef}
        src={fullUrl}
        poster={`${MEDIA_BASE_URL}${thumbnailUrl}`}
        loop
        muted={compact}
        playsInline
        controlsList="nodownload nofullscreen noremoteplayback"
        disablePictureInPicture
        draggable={false}
        onDragStart={(e) => e.preventDefault()}
        onTimeUpdate={handleTimeUpdate}
        onContextMenu={(e) => e.preventDefault()}
        className="w-full rounded-lg"
        style={{ pointerEvents: "none" }}
      />

      {/* Controls */}
      <div className="mt-2 flex items-center gap-2 text-xs">
        <button
          onClick={togglePlay}
          className="px-2 py-1 rounded bg-surface hover:bg-surface-hover"
        >
          {isPlaying ? "⏸" : "▶"}
        </button>

        {/* Progress bar */}
        <div
          className="flex-1 h-1.5 bg-surface rounded cursor-pointer"
          onClick={handleSeek}
        >
          <div
            className="h-full bg-accent rounded"
            style={{ width: `${(currentTime / duration) * 100}%` }}
          />
        </div>

        <span className="text-muted tabular-nums">
          {formatTime(currentTime)} / {formatTime(duration)}
        </span>

        {!compact && (
          <button
            onClick={cycleSpeed}
            className="px-1.5 py-0.5 rounded bg-surface hover:bg-surface-hover text-[10px]"
          >
            {playbackRate}x
          </button>
        )}
      </div>
    </div>
  );
}
```

## 실행 방식

### Option A: Subagent-Driven (추천)

각 태스크를 독립된 서브에이전트에게 위임하는 방식.

- 태스크 하나가 끝나면 결과를 리뷰한 뒤 다음 태스크 진행
- 의존성이 없는 태스크는 **병렬 실행** 가능 (예: Task 8, 10, 11 동시 진행)
- 각 에이전트는 해당 태스크의 파일만 수정 → 충돌 최소화
- 태스크 간 리뷰 체크포인트가 있어서 문제를 조기 발견

**장점:** 빠름, 병렬 가능, 태스크 간 격리
**단점:** 컨텍스트 전환 비용

### Option B: Inline Execution

현재 대화 세션에서 직접 순차 실행하는 방식.

- 18개 태스크를 위에서부터 하나씩 실행
- 주요 마일스톤(Task 7, 14, 17)에서 체크포인트 리뷰
- 하나의 세션에서 전체 컨텍스트 유지

**장점:** 전체 흐름 파악 용이, 컨텍스트 일관성
**단점:** 병렬 불가, 세션이 길어짐
